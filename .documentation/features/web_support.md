# AnsiX Features


## Web Support

**AnsiX** runs also on Flutter web applications! üåçüöÄ

Supports ANSI formatting only on **Chrome** _(and other Chromium-based browsers)_, **Opera** and **Edge**.
On other browsers all text will be printed unformatted.

Just make sure you have set the `ColorFormat` to `RGB`,
because web browsers don't support ANSI color codes.

```dart
AnsiX.colorFormat = ColorFormat.rgb;
```
